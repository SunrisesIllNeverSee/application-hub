## Products

| F6S | Program directory, apply links | No question reuse, no ranking, no AI |
| Submittable | Program intake for funders | Funder-side only |

| Startup Genome | Ecosystem data | Research tool, not application tool |
| Airtable / Notion templates | DIY trackers shared on Reddit | Manual, no intelligence, no data layer |

## White Space (Confirmed)

No product combines:
1. Cross-category program hub (accelerators + grants + fellowships + jobs)
2. Shared question archive with deduplication
3. Applicant-side workflow (profile answers → per-program drafts)
4. Program ranking by value/ROI
5. MCP exposure for AI assistants
