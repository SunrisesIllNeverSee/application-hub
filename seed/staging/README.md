# Staging Seed Lane

This folder is the holding area for candidate programs and orgs that may become part of the active seed set.

It exists to keep Firecrawl-assisted discovery separate from the production schema until the data is worth promoting.

Use this folder for:

- watchlists
- lightweight research outputs
- reviewed candidate rows
- staging helpers that should not be treated as canonical seed truth yet

Do not put final production seed SQL here.

Production-ready seeds belong in `seed/programs/`.
